Star Atlas Auto Re-supply

We have published an in-browser utility script for interaction with the Star Atlas SCORE minigame. This allows a user to automatically re-supply their ships without user interaction. The script is completely open and the user has full control, rather than relying on a third party website.

Town Star Auto-sell

We have published an in-browser utility script for interaction with the game Town Star. This provides several convenience features to enhance user experience such as auto-sell, town location discovery, and craft production statistics. This script was the first free, public, open source tool of its kind; and although it is no longer maintained, several forks have been created.
